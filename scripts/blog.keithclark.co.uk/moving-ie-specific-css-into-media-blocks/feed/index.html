]]> By: Keith Clark http://blog.keithclark.co.uk/moving-ie-specific-css-into-media-blocks/#comment-3393 Keith Clark Fri, 09 Nov 2012 10:05:12 +0000 http://blog.keithclark.co.uk/?p=321#comment-3393 A better solution for the above would be:

@media screen and (min-width:640px), screen/9 {
    body {
        background: green;
    }
}

This allows all non-IE browsers to render the styles and keeps media query support in IE9/10. It also creates a pass-through filter for IE6/7 but we’re still stuck with IE8 ignoring the entire block.
