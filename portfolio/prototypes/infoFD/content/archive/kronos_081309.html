The Kronos 4500 terminal is a high-speed network-centric badge terminal designed to capture and manage labor data. It features a large 3 x 4" backlit display that provides clear readability and a secure, intelligent ATM-like display. The terminal can accommodate very large volumes of employees, shifts, current and future schedules, as well as transaction data. It is our most sophisticated badge terminal, offering the following features out of the box:

  • Simple punch
  • Review punches
  • Display last punch
  • Display current schedules
  • Restrict punches
  • Display next schedule
  • Time since last punch
  • Display status
  • Totals
  • Check PTO balance
  • Approve timecard
  • View overtime
  • View vacation time
  • View sick time
